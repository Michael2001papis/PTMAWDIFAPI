    אודות האפליקציה

    הפרויקט נבנה בהתאם לסילבוס המכללה – הנה הטכנולוגיות והאמצעים שבהם השתמשתי:

    • HTML5 – תגיות סמנטיות, טפסים, מבנה נגיש
    • CSS3 – Flexbox, Transitions, Media Queries, משתנים
    • JavaScript – DOM, Event Listeners, JSON
    • Web Storage API – localStorage לאחסון מקומי
    • Fetch API – יבוא נתונים מ-JSONPlaceholder
    • Vite – בנייה ואופטימיזציה

    לא רק מימוש – גם הבנה והסבר של כל רכיב.

    ברוכים הבאים!

    מדריך קצר לפני שמתחילים:

    1. הוספת משימה – הזן תיאור ולחץ "הוסף משימה" (או Enter)
    2. התחברות – לחץ "התחבר / הירשם" ליצירת חשבון (שם מלא, טלפון, סיסמה) או להתחברות
    3. עריכה ומועדפים – רק משתמשים מחוברים יכולים לערוך משימות (✎) ולשמור למועדפים (⭐)
    4. חיפוש – השתמש בשורת החיפוש למציאת משימות לפי טקסט או תאריך
    5. סינון – הכל, פעילות, הושלמו או מועדפים
    6. מיון – מיון לפי תאריך יעד
    7. הגדרות – ⚙ לשינוי צבע וגודל
